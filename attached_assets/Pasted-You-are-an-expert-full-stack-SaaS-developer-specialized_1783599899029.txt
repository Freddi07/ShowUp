You are an expert full-stack SaaS developer specialized in AI tools for Norwegian small businesses. Build a complete, production-ready SaaS application called **LokalRadar**.

**Product vision (combines both user ideas):**
LokalRadar is an all-in-one AI tool for Norwegian local businesses (småbedrifter, butikker, restauranter, håndverkere, frisører etc.). It has two tightly integrated modules:

1. **KonkurrentOvervåking**  
   Users add competitors by pasting website URL and/or Google Business Profile URL. The system scrapes the website for prices/offers, fetches Google reviews/rating via Places API, detects changes (price drops, new reviews, new offers), and uses Claude to analyze what the change means and suggest concrete actions for the user.

2. **Markedsføringsassistent**  
   AI generates ready-to-use Google Posts / sosiale medier-innlegg, writes professional replies to Google reviews, and gives SEO/website improvement suggestions. The two modules feed each other (e.g. competitor insights can inspire post ideas).

**Unified experience:** One login, one dashboard, central AI chat that has context from both modules, and smart alerts that combine monitoring + marketing opportunities.

**Tech stack (use exactly this):**
- Next.js 15 (App Router) + TypeScript + Tailwind CSS + shadcn/ui + Radix UI
- Supabase (Auth + Postgres + Realtime + Storage) – this is the recommended stack for AI SaaS in 2026
- Anthropic SDK (@anthropic-ai/sdk) – Claude 3.5 Sonnet or Opus
- Google Places API (New) for reviews and rating
- Scraping: axios + cheerio (primary). Send HTML to Claude for intelligent structured extraction (prices, offers, etc.)
- Charts: Recharts
- Forms: react-hook-form + zod
- Icons: lucide-react
- Make it a full PWA (web app manifest + installable on mobile) so it feels like a native mobile app

**Critical requirements:**
- **All user-facing text must be in natural, professional Norwegian (bokmål).** Headlines, buttons, labels, toasts, AI outputs, emails – everything. Generate high-quality Norwegian from the start.
- Modern, clean, trustworthy design (professional blues/greens, good typography, excellent mobile experience).
- Freemium model: Free tier (1 competitor + limited AI generations), Pro (149 kr/mnd), Bedrift (399 kr/mnd – multiple locations + priority support).

**Database schema (create via Supabase):**
- businesses (user's own business info)
